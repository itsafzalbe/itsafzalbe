## About me

I'm a backend-focused computer science student at Webster University in Tashkent. I spend most of my time designing and building backend systems — APIs, databases, task queues, deployment pipelines — and thinking about what makes software reliable at scale.

My interests sit at the intersection of **backend engineering**, **fintech infrastructure**, and **cloud-native systems**. I like working close to the infrastructure layer: understanding how things actually run, not just how they're written.

Outside of code I think a lot about system design, distributed architecture patterns, and how consumer software is evolving in Central Asian markets.

## What I'm working on

- 🏗️ **CartIZ** — A full e-commerce backend built with Django REST Framework. 30+ table schema, Redis caching, Celery task queue, JWT auth, and a complete promotions/discounts engine. Deployed with Nginx + Gunicorn + PgBouncer.

- 🛍️ **Wearly** — A secondhand clothing marketplace for Uzbekistan. Starting as a Telegram-based MVP, evolving into a full DRF + React web platform and eventually React Native mobile. Think Vinted, but built for the Central Asian market.

- 🖥️ **AVZALBEK OS** — A terminal-style developer portfolio. FastAPI + MongoDB backend, React/Vite frontend, 6 themes (Tokyo Night default), full terminal engine with command parsing, GUI mode with glassmorphism, and a few ASCII easter eggs.

- 🔍 **SearchEngine** — A C++ search engine with an HTML parser built on lexbor, URL frontier, and link extraction. Written in Uzbek language context. A grad-school application project and a deep dive into systems programming.
